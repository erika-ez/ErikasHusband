# Erika's Husband

Relaunching Erika's Husband's ;) blog era with **Jekyll**.

---

## Running Locally

1. Install Ruby and RubyGems  
2. Run: `gem install jekyll bundler`  
3. Then: `bundle install`
4. `bundle exec jekyll serve`
5. Preview at: [http://127.0.0.1:4000/ErikasHusband/](http://127.0.0.1:4000/Erikasblog/)
